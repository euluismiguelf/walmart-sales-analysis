# 🛒 Walmart Sales Analysis | Excel + Power BI

## 📋 Descripción
Análisis de ventas semanales de 45 tiendas Walmart (2010-2012) 
con más de 6,400 registros. El proyecto explora el impacto de 
factores estacionales y macroeconómicos en las ventas.

## 🎯 Preguntas de Negocio
- ¿Qué tiendas generan más ventas?
- ¿Las semanas festivas impactan las ventas?
- ¿Factores como temperatura y desempleo afectan el consumo?

## 🔍 Principales Hallazgos
- La tienda #20 lidera con $301.4M en ventas totales
- Las semanas festivas generan un 7.8% más de ventas
- Pico estacional claro en Noviembre-Diciembre
- El desempleo tiene impacto mínimo (R²=0.01) — consistente
  con la estrategia de precios bajos de Walmart
